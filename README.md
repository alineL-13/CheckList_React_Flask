# CheckList
A full-stack project built to practice front-end and back-end integration, data persistence, and component organization.
The application allows users to create tasks, sort them by due date, update their status, and remove items.

# Features
- Create tasks with title, description, due date and status (to do, doing, done)
- Validation for required fields
- Automatic sorting by due date
- Quick status updates
- Delete individual tasks
- Clear all tasks marked as "done"
- Data persistence

# Technologies Used
Frontend:
- HTML
- CSS
- JavaScript
- React
- TailwindCSS

Backend:
- Flask
- SQLAlchemy
- SQLite

# Running the Project:
Backend:
- cd backend
- pip install -r requirements.txt
- flask run

Frontend:
- cd frontend
- npm install
- npm run dev
